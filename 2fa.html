TweeFactor verificatie

Naast een sterk en uniek wachtwoord per online dienst bestaat er nog een tweede belangrijke beveiligingsoptie: tweeFactor verificatie. 
Maak hier gebruik van!!!

Wat is tweefactorauthenticatie?
Twee-factor of multi-factor authenticatie is een extra inlogcode voor een account - een tweede verdedigingslinie voor jouw gevoelige info. Er zijn een aantal manieren waarop je deze korte, secundaire inlogcodes kunt krijgen. 
Het basisidee is dat een enkel wachtwoord voor je belangrijke accounts eenvoudig niet genoeg is. Als jouw wachtwoord wordt geraden, of hackers een database stelen met jouw aanmeldingsgegevens in platte tekst, is je account een ‘Sitting Duck’. TweefactorAuthenticatie probeert dat probleem op te lossen door een secundaire code te vragen (meestal zes tekens lang en gegenereerd door een smartphone-app) voordat je toegang kunt krijgen tot je account. Op die manier moet, zelfs als een hacker je wachtwoord heeft, hij nog steeds een secundaire code kraken. Deze code hefeft een geldigheidsduur van bv 30 seconden. Dit maakt het wel heel moeilijk om te kraken!

Let op, 2FA-codes via sms ontvangen is raden we af daar de code mogelijk kan worden onderschept door hackers, zoals onlangs is aangetoond door onderzoekers voor positieve technologieën. Dat is de reden waarom het gebruik van een software- of hardware-gebaseerde oplossing op een apparaat dat u bezit de voorkeur heeft.

Elke service die de standaard 2FA-aanpak ondersteunt, werkt met alle onderstaande apps, inclusief de meeste gangbare websites en services.
